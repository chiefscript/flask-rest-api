# Simple Flask REST API

REST API for a library

## Setup

**Requirements**

* Python 3+
* Pipenv


**Start virtual environment & install requirements**

    $ pipenv shell
    $ pipenv install
    
**Other commands**

Listing all routes
    
    $ python manage.py routes
    $ flask routes
